# Django Todo App

A simple Todo application built using Django.

## Features
- Add tasks
- Delete tasks
- Admin panel support
- SQLite database

## Tech Stack
- Python
- Django
- HTML, CSS
- SQLite

## Run Locally
```bash
git clone https://github.com/vardanvishnu/django-todo.git
cd django-todo
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
